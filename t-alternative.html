Figuring out which LebreOffice application to use

Context

LibreOffice applications correlate seamlessly to those in the Microsoft Office Suite. So, just about any task Microsoft Office can be used to complete can also be completed using LibreOffic.

Prerequisites

  • LibreOffice 4.3

Steps

  • Use LibreOffice Writer to drafts documents, much in the same way Microsoft Office Word does See Figure 1 Below
  • Use LibreOffice Calc to organize spreadsheets, much in the same way Microsoft Office Excel does See Figure 2 Below
  • Use LibreOffice Impress to create presentations, much in the same way Microsoft Office Powerpoint does See Figure 3 Below
