Time to setup LibreOffice

Context

To get started using LibreOffice 4.3, we must first download and install the application.

Pre-requisites

  • Computer
  • Internet access

Steps

  1. Go to LibreOffice's website, www.libreoffice.org
  2. Click the 'Download' drop-down menu, and select 'LibreOffice Fresh' See Figure 1 Below
  3. Download LibreOffice Version 4.3.0 by clicking the appropriate link See Figure 2 Below

    Be sure to read this page and carefully choose the correct download for your system

  4. Save the download to your computer when prompted
  5. Open LibreOffice's installation wizard after the download has completed by clicking run
  6. Read through the installation wizard's prompts, successfully downloading and install Libre Office 4.3 See Figure Below

Examples

Click 'LibreOffice Fresh' under the 'Download' drop-down menu (Figure 1)
